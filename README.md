## Lessons Learned  

Con este proyecto aprendí a utilizar React, Context y Firebase

## Ejecutar localmente

Clona el proyecto con

~~~bash  
  git clone https://github.com/dander93/CoderHouse-React.git
~~~

Ingresa en la carpeta del proyecto 

~~~bash  
  cd dander-ecommerce
~~~

Instala las dependencias  

~~~bash  
npm install
~~~

Inicia el servidor  

~~~bash  
npm run start
~~~
